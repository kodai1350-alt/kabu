## 分析指標（J-Quants MCP + TradingView MCP）
1. RSI(14)：30以下=過売り買いシグナル、70以上=過買い売りシグナル
2. MACD：ゴールデンクロス/デッドクロスの確認
3. ボリンジャーバンド(20,2)：±2σでのリバーサル狙い
4. 移動平均：5日・25日・75日・200日の配置確認
5. 出来高：直近平均の1.5倍以上 = シグナル強化
6. VIX（恐怖指数）：30以上は相場全体のリスクオフ

## エントリールール（全条件を満たす時のみ）
- [ ] マクロ環境スキャン: リスクオン or 中立
- [ ] 企業精密分析: 総合スコア15点以上
- [ ] RSI(14): 35〜65の範囲
- [ ] 株価が200日移動平均線より上にある
- [ ] 直近の出来高が平均以上
